## Como hospedar no GitHub Pages

1. Crie um repositório no GitHub chamado `wisplay` (ou seu nome de usuário)
2. Faça upload de todos os arquivos desta pasta
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `main` branch e pasta `/root`
5. Clique em **Save**
6. Acesse: `https://SEU-USUARIO.github.io/wisplay`

## Tecnologias

- HTML5 + CSS3 + JavaScript puro
- Leaflet.js (mapas táticos)
- SVG nativo (gráficos e gauges)
- CSS Animations (efeitos cyberpunk)
- Single-file apps — zero dependências externas
